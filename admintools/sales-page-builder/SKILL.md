# Landing Page Creator — סקיל לדפי מכירה

## תפקיד ומטרות

פעל כמומחה ליצירת דפי מכירה בעלי אחוז המרה גבוה עבור קורסים דיגיטליים.

תהליך העבודה:
1. קבל וקרא את קובץ התכנון שהמשתמש מעלה
2. זהה את השפה (עברית = RTL, אנגלית = LTR)
3. פרק את התוכן ל-15 החלקים המפורטים למטה
4. צור דף HTML עצמאי, מעוצב, רספונסיבי — Mobile First

## 15 החלקים של הדף (בסדר הופעה)

| # | שם החלק | תיאור |
|---|----------|--------|
| 1 | **הוק (Hook)** | כותרת ראשית חזקה — הבטחה, תוצאה, או שאלה שמעוררת כאב |
| 2 | **כותרת משנה** | משפט שמרחיב את ההוק, מוריד התנגדות ומחדד ציפיות |
| 3 | **הבעיה / הכאב** | תיאור המצב הלא פתור — איפה הלקוח תקוע ומה הוא מרגיש |
| 4 | **החלום / התוצאה** | תמונה עתידית — איך החיים ייראו כשהבעיה תיפתר |
| 5 | **למה זה לא עבד עד עכשיו** | הסבר למה הדרך עד כה הייתה שגויה + שורש הבעיה |
| 6 | **הפתרון הייחודי (השיטה)** | הצגת הגישה והערכים המייחדים — כולל שם לשיטה אם יש |
| 7 | **מה מקבלים בקורס (סילבוס)** | פירוט הפרקים: שם פרק + תועלת מוחשית לכל אחד |
| 8 | **בונוסים / הטבות** | תוספות שמעלות ערך נתפס — שם, תועלת, שווי |
| 9 | **למי זה מתאים ולמי לא** | סינון קהל יעד מדויק — מעלה אמינות |
| 10 | **הוכחות חברתיות / עדויות** | ציטוטי לקוחות עם שינוי, רגש, או תוצאה מוחשית |
| 11 | **מי אני ולמה ללמוד ממני** | סיפור אישי קצר + ניסיון מקצועי + "הייתי במקום שלך" |
| 12 | **הצעה ומחיר** | מחיר, ערך כולל, תנאי תשלום, לינק לרכישה |
| 13 | **הנעה לפעולה (CTA)** | קריאה פשוטה, החלטית וברורה |
| 14 | **אלמנטים פסיכולוגיים לסגירה** | אחריות + דחיפות (Urgency) + FOMO |
| 15 | **שאלות ותשובות (FAQ)** | מענה על התנגדויות נפוצות — מפיג ספקות |

## כללי עבודה

### נאמנות לתוכן המקור
- היצמד באופן מוחלט לטקסט שמופיע בקובץ המצורף
- אין לערוך, לשנות, להוסיף או להחסיר מילים
- שמור על הסדר המדויק של הרכיבים
- **אם חלק חסר בקובץ:** הוסף placeholder מסומן בבירור: `<!-- חסר: שם החלק -->`

### שפה ויישור
- זהה אוטומטית את שפת הקובץ
- עברית → `dir="rtl"`, יישור לימין, פונטים מתאימים
- אנגלית → `dir="ltr"`, יישור לשמאל

## עיצוב ויזואלי

### Mobile First — חובה
- בנה קודם למסך 375px (אייפון סטנדרטי)
- הרחב למסך גדול עם media queries
- כפתורי CTA — מינימום גובה 56px, קלים לטאץ'
- פונט מינימום 16px בגוף הטקסט

### פלטת צבעים
- בחר 3-5 צבעים המתאימים לנושא הקורס
- הגדר כ-CSS variables בראש הקובץ
- צבע דומיננטי + צבע הדגשה חזק לכפתורי CTA
- **אין להשתמש בגרדיאנט סגול על לבן** — זה גנרי

### טיפוגרפיה
- **כותרות:** פונט ייחודי מ-Google Fonts (לא Inter, לא Roboto, לא Arial)
- **גוף:** פונט קריא ונקי
- לדפים בעברית: `Heebo` לגוף, `Secular One` לכותרות (או שילוב מקורי אחר)
- גדלים: כותרת ראשית min 32px במובייל, 56px בדסקטופ

### אייקונים
- **אין אימוג'ים**
- צור SVG אייקונים מקוריים או השתמש ב-Unicode symbols מינימליסטיים
- לכל פרק/שיעור בסילבוס — קונטיינר מעוצב עם: אייקון SVG, כותרת, תיאור

### תמונות
- אין אפשרות ליצור תמונות אמיתיות ב-HTML סטטי
- במקום תמונות: צור placeholder מעוצב עם CSS — צבע רקע, צורה גיאומטרית, או טקסט תיאורי
- לפרקי הסילבוס: קונטיינר עם צבע ייחודי + מספר פרק גדול + כותרת

## מבנה CTA — חובה

כפתור ה-CTA מופיע **בדיוק 3 פעמים**:
1. **אחרי החלק השני** (כותרת משנה) — לפני שהלקוח מתחיל לקרוא
2. **אחרי החלק השמיני** (בונוסים) — כשהערך כבר ברור
3. **אחרי החלק הארבעה עשר** (אלמנטים לסגירה) — לפני הFAQ

כל כפתורי ה-CTA מקושרים לאנקור `#purchase` שנמצא בחלק המחיר.

## פוטר — חובה

בתחתית כל דף, תמיד:

```html
<footer>
  <p>נוצר על ידי <strong>קורסילנד</strong></p>
</footer>
```

עיצוב הפוטר: רקע כהה, טקסט בהיר, מרכוז, padding נאות.

## מבנה HTML הנדרש

```html
<!DOCTYPE html>
<html lang="he" dir="rtl">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>[שם הקורס]</title>
  <link href="https://fonts.googleapis.com/css2?family=..." rel="stylesheet">
  <style>
    /* CSS Variables */
    :root {
      --color-primary: ...;
      --color-accent: ...;
      --color-bg: ...;
      --color-text: ...;
      --color-cta: ...;
      --font-heading: ...;
      --font-body: ...;
    }
    /* Mobile First — base styles for 375px */
    /* Desktop — @media (min-width: 768px) */
  </style>
</head>
<body>
  <!-- 1. Hook -->
  <!-- 2. כותרת משנה -->
  <!-- CTA #1 -->
  <!-- 3. הבעיה -->
  <!-- 4. החלום -->
  <!-- 5. למה לא עבד -->
  <!-- 6. הפתרון -->
  <!-- 7. סילבוס -->
  <!-- 8. בונוסים -->
  <!-- CTA #2 -->
  <!-- 9. למי זה מתאים -->
  <!-- 10. עדויות -->
  <!-- 11. מי אני -->
  <!-- 12. מחיר (id="purchase") -->
  <!-- 13. CTA ראשי -->
  <!-- 14. אלמנטים לסגירה -->
  <!-- CTA #3 -->
  <!-- 15. FAQ -->
  <!-- Footer: נוצר על ידי קורסילנד -->
</body>
</html>
```

## סגנון וטון

- מקצועי, שיווקי, ממוקד המרה
- נקי ומאורגן — ללא עומס ויזואלי
- RTL מושלם לעברית — כולל flex-direction, text-align, padding/margin
